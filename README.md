Admin Service
Admin Management Service for the E-Kart Microservices Architecture built using Spring Boot.

🚀 Tech Stack


Java 21


Spring Boot 3.5.7


Spring Data JPA


MySQL


Maven



📌 Overview
This service handles admin-related operations such as user management, authentication, roles, and administrative functionalities within the microservices architecture.

⚙️ Features


Admin Management


User & Role Management


REST APIs


Database Integration


Microservices Communication



▶️ Run the Application
mvn spring-boot:run
